CANNONBALL — audio drop folder
==============================

Drop your sound files here using these EXACT base names (the game loads .ogg,
then .wav, then .mp3 for each). A missing file is fine — its slot stays silent.

BALL (accuracy matters — real recordings; Freesound.org, filter: CC0):
  kick     — a header / hard hit on the ball   (search: "football kick","ball hit")
  bounce   — ball bouncing on the pitch         (search: "ball bounce","leather bounce")
  land     — ball / bean landing thud on grass  (search: "thud grass","soft impact")

MATCH:
  cannon   — kickoff cannon boom                (search: "cannon fire","firework launch")
  whistle  — referee whistle                    (search: "referee whistle")
  elim     — a player eliminated                (search: "whoosh out","poof")
  goal     — GOLDEN BOOT goal scored            (search: "goal net","crowd goal")
  tick     — tense countdown blip               (search: "beep tense","clock tick")
  save     — free-save punt                     (search: "punt","kick clear")

CROWD (you're supplying these):
  crowd    — ambient stadium crowd (LOOPS)      — a steady murmur/ambience
  cheer    — crowd cheer swell (goal/elim)

UI / WORLD:
  click    — menu / button click                (Kenney UI packs are fine, CC0)
  bang     — floodlights switch on at nightfall (search: "power on","electric hum on")

MUSIC:
  music    — looping lo-fi background bed        (Freesound CC0 "lo-fi loop", or
                                                  OpenGameArt; Incompetech = CC-BY)

LICENSING: prefer CC0 (no attribution). If you use a CC-BY sound, add a line to
CREDITS.txt (sound name, author, source URL, license). Avoid CC-BY-NC.
Controls in-game: M = mute toggle. Volume is persisted.
